# LLM+K8S Utils

A collection of LLM and Kubernetes utilities. Each subdirectory is a
self-contained tool — see its own `README.md` for setup and usage.

## Contents

- [`openai-sdk-client/`](openai-sdk-client/) — OpenAI-SDK chat/responses test
  clients for an OpenAI-compatible dev server.
- [`s3-explorer/`](s3-explorer/) — interactive, read-only inspector for
  S3-compatible object stores (MinIO/Ceph/self-hosted, and AWS S3).
